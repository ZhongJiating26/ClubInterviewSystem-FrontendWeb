  1. 分配角色

  POST /api/v1/auth/assign-role

  请求：
  {
    "user_id": 1,
    "role_id": 4,        // 角色ID
    "club_id": null
  }

  响应：
  {
    "detail": "角色分配成功",
    "user_role": {
      "id": 1,
      "user_id": 1,
      "role_id": 4,
      "club_id": null
    }
  }

  2. 撤销角色

  POST /api/v1/auth/revoke-role

  请求：
  {
    "user_id": 1,
    "role_id": 4,
    "club_id": null
  }

  响应：
  {"detail": "角色撤销成功"}
